# PRD Snapshot: Handoff Studio MVP

## Purpose
Launch a premium, conversion-focused service studio website quickly while preserving founder privacy and keeping brand identity configurable.

## MVP Objectives

- Clear value proposition on first screen
- Low-friction lead capture via intake (no sales call required)
- Strong trust signals: proof tiles, scope clarity, turnaround clarity
- Static deployment for speed (Cloudflare Pages)

## MVP Pages

- Home: value proposition, services, process, proof, CTA
- Services: productized packages with scope boundaries
- Process: Send -> Plan -> Deliver
- Contact: intake CTA, email fallback, privacy note

## Non-Goals (MVP)

- CMS/blog
- user accounts
- payments
- complex workflow automation backend

## Core System Ideas

- Config-driven branding/content from one file
- Static-first architecture with Django-ready extension path
- Privacy-by-design public identity
